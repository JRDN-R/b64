GSUS Media ⇄ Base64

Upload one or multiple media files and download them as a single custom .gsus Base64 bundle. Upload that .gsus file later to rebuild and view the media in a new tab.

1. Media to .gsus

Choose images, videos, or audio files. Multiple files are packaged into one large Base64 .gsus file.

2. .gsus back to media

Upload a .gsus file made by this tool. It can contain one file or a whole media bundle.

Decoded media preview will show here.

The .gsus extension is custom. The file is still readable text, but it contains a JSON media bundle with Base64 data URIs inside. Large videos can become very large because Base64 adds roughly one-third more size.
