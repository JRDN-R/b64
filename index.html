Image ⇄ Base64 URI

Upload an image to turn it into a downloadable Base64 data URI file. Then upload that saved file later to rebuild the image and open it in a clean full-window tab.

1. Image to Base64 URI file

Choose a PNG, JPG, WebP, GIF, SVG, or other browser-readable image file.

2. Base64 URI file back to image

Upload a .b64uri, .json, or plain .txt file containing a Base64 image data URI.

Decoded image preview will show here.

Format made by this page: a readable JSON file ending in .b64uri that contains the original file name, MIME type, and full data:image/...;base64,... URI. The decoder also accepts a raw data URI pasted into a text file.
