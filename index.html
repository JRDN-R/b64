GSUS Media ⇄ Base64

Upload one or more images, videos, or audio files and download them as one custom .gsus Base64 bundle. Then upload that .gsus file later to rebuild and view the media.

1. Media to .gsus

Choose images, videos, or audio files. Multiple files become one large .gsus file.

Selected files will show here after processing.

2. .gsus back to media

Upload a .gsus file made by this tool. The iPhone picker is intentionally not restricted so custom .gsus files are selectable.

Decoded media preview will show here.

Blunt limitation: Base64 makes files about one-third larger. Big videos can choke mobile Safari or fail from memory limits.
